# IPL Cricket Analysis & Prediction Project

## Project Overview
This project focuses on analyzing Indian Premier League (IPL) cricket data using Python.  
The analysis extracts meaningful insights from match data and visualizes key trends using graphs.

This project was completed as part of the **GUVI Data Science Course**

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
- `matches.csv`  
Contains IPL match-level data used for analysis.

## Analysis & Visualizations
The project includes:
- Team-wise performance analysis  
- Match outcome trends  
- Visualizations using graphs (png files included)
